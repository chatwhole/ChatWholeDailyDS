## Key Concepts Summary

### Generative AI Model Types

| Type | Examples | Use Case |
|------|----------|----------|
| LLM | GPT-4, Llama, Claude | Text generation |
| Diffusion | Stable Diffusion, DALL-E | Image generation |
| GAN | StyleGAN, ProGAN | Image synthesis |
| VAE | VQ-VAE, β-VAE | Image generation, anomalies |
| Multimodal | GPT-4V, Gemini | Text + Image |
| Audio | Whisper, Bark | Speech/Text |

### RAG Architecture

```
Query → Embedding → Vector Search → Retrieve Docs → Context → LLM → Response

Components:
- Embedding Model: OpenAI, Cohere, BGE
- Vector DB: Pinecone, Chroma, Weaviate, Milvus
- LLM: GPT-4, Claude, Llama
```

### Fine-Tuning Approaches

| Method | Description | When to Use |
|--------|-------------|-------------|
| Full FT | Update all parameters | Large dataset, high compute |
| LoRA | Low-rank adaptation | Medium dataset |
| QLoRA | Quantized LoRA | Low compute |
| Prompt Tuning | Learnable prompts | Minimal compute |
| ICL | In-context learning | No training needed |

### Agent Components

```
LLM + Tools + Memory + Planning = Agent

Tools: API calls, code execution, web search
Memory: Short-term (context), Long-term (vector DB)
Planning: ReAct, Chain-of-thought, Tree-of-thought
```

### Text-to-Image Models

| Model | Type | Key Feature |
|-------|------|-------------|
| Stable Diffusion | Diffusion | Open source |
| DALL-E 3 | Diffusion | GPT-4 integration |
| Midjourney | Diffusion | Artistic quality |
| Flux | Flow matching | High fidelity |
| Imagen | Diffusion | Google |

### Key Concepts

| Concept | Description |
|---------|-------------|
| Temperature | Controls randomness (0=deterministic, 1=creative) |
| Top-p | Nucleus sampling |
| Context Window | Maximum input length |
| Tokens | Subword units |
| Embedding | Dense vector representation |
| Attention | Dynamic weighting of inputs |
