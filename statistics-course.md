## Key Concepts Summary

### Measures of Central Tendency

| Measure | Formula | When to Use |
|---------|---------|-------------|
| Mean | Σx/n | Symmetric data |
| Median | Middle value | Skewed data |
| Mode | Most frequent | Categorical data |
| Weighted Mean | Σ(w·x)/Σw | Different importance |
| Geometric Mean | (Πx)^(1/n) | Growth rates |
| Harmonic Mean | n/Σ(1/x) | Rates, ratios |

### Measures of Spread

| Measure | Formula | Description |
|---------|---------|-------------|
| Range | max - min | Simple spread |
| IQR | Q3 - Q1 | Middle 50% |
| Variance | Σ(x-x̄)²/n | Average squared deviation |
| Std Dev | √variance | Same units as data |
| CV | (std/mean)×100 | Relative variability |

### Correlation Methods

| Method | Use Case | Range |
|--------|----------|-------|
| Pearson | Linear, continuous | [-1, 1] |
| Spearman | Monotonic, ordinal | [-1, 1] |
| Kendall | Small samples, ordinal | [-1, 1] |
| Point Biserial | Binary + continuous | [-1, 1] |
| Phi | Binary + binary | [-1, 1] |
| Cramer's V | Categorical | [0, 1] |

### Probability Distributions

| Distribution | Use Case |
|-------------|----------|
| Bernoulli | Single trial, binary outcome |
| Binomial | n trials, binary outcome |
| Poisson | Count data, rare events |
| Normal | Continuous, symmetric |
| Exponential | Time between events |
| Chi-Square | Categorical data, goodness of fit |
| t-distribution | Small samples, mean testing |
| F-distribution | Variance comparison |

### Hypothesis Testing Framework

```
1. State H₀ and H₁
2. Choose significance level (α = 0.05)
3. Calculate test statistic
4. Find p-value
5. Decision: reject H₀ if p < α
```

### Common Tests

| Test | When to Use |
|------|-------------|
| One-sample Z | Known σ, large n |
| One-sample T | Unknown σ, small n |
| Two-sample T | Compare two means |
| Paired T | Before/after |
| Chi-Square GoF | Categorical fit |
| Chi-Square Independence | Two categorical variables |
| ANOVA | Compare 3+ means |
