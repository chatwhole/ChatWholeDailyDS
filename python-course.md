## Key Concepts Summary

### Data Types

| Type | Example | Mutable | Ordered |
|------|---------|---------|---------|
| int | 42 | No | N/A |
| float | 3.14 | No | N/A |
| str | "hello" | No | Yes |
| list | [1,2,3] | Yes | Yes |
| tuple | (1,2,3) | No | Yes |
| dict | {"a":1} | Yes | Yes (3.7+) |
| set | {1,2,3} | Yes | No |

### List Comprehensions

```python
# Basic
squares = [x**2 for x in range(10)]

# With condition
evens = [x for x in range(10) if x % 2 == 0]

# Nested
matrix = [[i*j for j in range(3)] for i in range(3)]
```

### Lambda Functions

```python
# Syntax
lambda arguments: expression

# Examples
square = lambda x: x**2
add = lambda x, y: x + y
```

### Decorators

```python
def my_decorator(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")
```

### Generators

```python
def my_generator(n):
    for i in range(n):
        yield i

# Lazy evaluation
gen = my_generator(5)
next(gen)  # 0
next(gen)  # 1
```

### Context Managers

```python
with open('file.txt', 'r') as f:
    content = f.read()

# Custom context manager
from contextlib import contextmanager

@contextmanager
def my_context():
    print("Enter")
    yield
    print("Exit")
```

### Error Handling

```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Error: {e}")
except Exception as e:
    print(f"Unexpected: {e}")
finally:
    print("Always runs")
```
