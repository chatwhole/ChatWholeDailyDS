## Key Concepts Summary

### Supervised Learning Algorithms

| Algorithm | Type | Strengths | Weaknesses |
|-----------|------|-----------|------------|
| Linear Regression | Regression | Simple, interpretable | Assumes linearity |
| Logistic Regression | Classification | Probabilistic output | Linear decision boundary |
| Decision Tree | Both | Interpretable, handles non-linear | Prone to overfitting |
| Random Forest | Both | Robust, handles missing data | Less interpretable |
| XGBoost | Both | High accuracy, fast | Requires tuning |
| SVM | Both | Effective in high dimensions | Slow on large datasets |
| KNN | Both | Simple, no training | Slow prediction |

### Unsupervised Learning Algorithms

| Algorithm | Purpose | Strengths | Weaknesses |
|-----------|---------|-----------|------------|
| K-Means | Clustering | Fast, simple | Assumes spherical clusters |
| DBSCAN | Clustering | Finds arbitrary shapes | Sensitive to parameters |
| Hierarchical | Clustering | Dendrogram visualization | Slow on large data |
| PCA | Dimensionality Reduction | Preserves variance | Linear only |
| t-SNE | Visualization | Preserves local structure | Non-parametric |

### Model Evaluation Metrics

| Metric | Use Case | Range |
|--------|----------|-------|
| Accuracy | Balanced classification | 0-1 |
| Precision | Minimize false positives | 0-1 |
| Recall | Minimize false negatives | 0-1 |
| F1 Score | Balanced precision/recall | 0-1 |
| AUC-ROC | Binary classification | 0-1 |
| MSE | Regression | 0-infinity |
| R² | Regression | 0-1 |

### Feature Engineering Techniques

| Technique | Description | When to Use |
|-----------|-------------|-------------|
| One-Hot Encoding | Convert categories to binary | Low cardinality |
| Label Encoding | Convert categories to integers | Tree-based models |
| StandardScaler | Normalize to mean=0, std=1 | Linear models |
| MinMaxScaler | Scale to 0-1 range | Neural networks |
| Log Transform | Reduce skewness | Skewed features |
| Polynomial Features | Create interaction terms | Non-linear relationships |

## Code Examples

### Linear Regression with scikit-learn

```python
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

# Sample data
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 5, 4, 5])

# Split data
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model
model = LinearRegression()
model.fit(X_train, y_train)

# Predict and evaluate
y_pred = model.predict(X_test)
print(f"R² Score: {r2_score(y_test, y_pred):.4f}")
print(f"MSE: {mean_squared_error(y_test, y_pred):.4f}")
```

### Random Forest Classifier

```python
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

# Load data
X, y = load_iris(return_X_y=True)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train model
clf = RandomForestClassifier(n_estimators=100, random_state=42)
clf.fit(X_train, y_train)

# Evaluate
y_pred = clf.predict(X_test)
print(classification_report(y_test, y_pred))
```

### XGBoost for Classification

```python
import xgboost as xgb
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# Generate data
X, y = make_classification(n_samples=1000, n_features=20, random_state=42)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Train XGBoost
model = xgb.XGBClassifier(
    n_estimators=100,
    max_depth=6,
    learning_rate=0.1,
    random_state=42
)
model.fit(X_train, y_train)

# Evaluate
y_pred = model.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
```

### K-Means Clustering

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# Generate sample data
X, y = make_blobs(n_samples=300, centers=4, random_state=42)

# Apply K-Means
kmeans = KMeans(n_clusters=4, random_state=42)
y_pred = kmeans.fit_predict(X)

# Plot results
plt.scatter(X[:, 0], X[:, 1], c=y_pred, cmap='viridis')
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], 
            s=300, c='red', marker='X')
plt.title('K-Means Clustering')
plt.show()
```

### Cross-Validation

```python
from sklearn.model_selection import cross_val_score
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

X, y = load_iris(return_X_y=True)

# 5-fold cross-validation
clf = RandomForestClassifier(n_estimators=100, random_state=42)
scores = cross_val_score(clf, X, y, cv=5, scoring='accuracy')

print(f"CV Scores: {scores}")
print(f"Mean Accuracy: {scores.mean():.4f} (+/- {scores.std() * 2:.4f})")
```

### Hyperparameter Tuning with GridSearch

```python
from sklearn.model_selection import GridSearchCV
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

X, y = load_iris(return_X_y=True)

# Define parameter grid
param_grid = {
    'n_estimators': [50, 100, 200],
    'max_depth': [None, 10, 20],
    'min_samples_split': [2, 5, 10]
}

# Grid search
grid_search = GridSearchCV(
    RandomForestClassifier(random_state=42),
    param_grid, cv=5, scoring='accuracy', n_jobs=-1
)
grid_search.fit(X, y)

print(f"Best Parameters: {grid_search.best_params_}")
print(f"Best Score: {grid_search.best_score_:.4f}")
```

### Feature Importance Visualization

```python
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_iris

X, y = load_iris(return_X_y=True)
feature_names = load_iris().feature_names

# Train model
clf = RandomForestClassifier(n_estimators=100, random_state=42)
clf.fit(X, y)

# Plot feature importance
importance = pd.Series(clf.feature_importances_, index=feature_names)
importance.sort_values().plot(kind='barh')
plt.title('Feature Importance')
plt.xlabel('Importance Score')
plt.show()
```

## Frequently Asked Questions

### What is the difference between supervised and unsupervised learning?
Supervised learning uses labeled data to train models (e.g., classification, regression), while unsupervised learning finds patterns in unlabeled data (e.g., clustering, dimensionality reduction).

### How long does it take to learn machine learning?
With 1-2 lessons per day, you can complete this 50-lesson course in 5-7 weeks. Building real-world expertise typically takes 6-12 months of practice.

### What is the best machine learning algorithm?
There is no single best algorithm. The choice depends on your data, problem type, and requirements. Start with simple models like logistic regression, then try ensemble methods like Random Forest and XGBoost.

### What prerequisites do I need for this course?
Basic Python programming and high school math are sufficient to start. We recommend completing our [Python course](https://chatwhole.com/learn/python/) and [Statistics course](https://chatwhole.com/learn/statistics/) first.

### What is overfitting and how do I prevent it?
Overfitting occurs when a model learns noise instead of patterns. Prevent it using cross-validation, regularization, pruning, early stopping, and getting more training data.

### What is the difference between Random Forest and XGBoost?
Random Forest uses bagging (parallel ensemble), while XGBoost uses boosting (sequential ensemble). XGBoost typically achieves higher accuracy but requires more tuning.

### How do I handle missing data in machine learning?
Options include deletion (listwise, pairwise), imputation (mean, median, mode, KNN, MICE), or using algorithms that handle missing values natively (XGBoost, LightGBM).

### What is feature engineering?
Feature engineering is the process of creating, transforming, and selecting input variables to improve model performance. It includes encoding, scaling, creating interactions, and domain-specific transformations.

### How do I deploy a machine learning model?
Use Flask or FastAPI to create an API, Docker for containerization, and cloud platforms (AWS, GCP, Azure) for hosting. See [Model Deployment](https://chatwhole.com/learn/machine-learning/22-model-deployment/) for step-by-step instructions.
