## Key Concepts Summary

### Supervised vs Unsupervised

| Type | Learning | Examples |
|------|----------|----------|
| Supervised | Labeled data | Classification, Regression |
| Unsupervised | No labels | Clustering, Dimensionality Reduction |
| Semi-supervised | Some labels | Self-training, Label propagation |
| Reinforcement | Reward signal | Game playing, Robotics |

### Regression Algorithms

| Algorithm | Use Case | Pros | Cons |
|-----------|----------|------|------|
| Linear Regression | Linear relationships | Simple, interpretable | Assumes linearity |
| Polynomial Regression | Non-linear | Flexible | Overfitting risk |
| Ridge Regression | Multicollinearity | Regularized | Biased |
| Lasso Regression | Feature selection | Sparse models | May select wrong features |
| Elastic Net | Both | Balanced | Two hyperparameters |

### Classification Algorithms

| Algorithm | Use Case | Pros | Cons |
|-----------|----------|------|------|
| Logistic Regression | Binary classification | Probabilistic output | Linear decision boundary |
| Decision Trees | General | Interpretable | Overfitting |
| Random Forest | General | Robust, accurate | Less interpretable |
| XGBoost | Competitions | High accuracy | Complex tuning |
| SVM | High-dimensional | Effective with kernels | Slow on large data |
| Naive Bayes | Text classification | Fast, works with small data | Independence assumption |
| KNN | Small datasets | Simple, no training | Slow prediction |

### Model Evaluation

| Metric | Use Case |
|--------|----------|
| Accuracy | Balanced classes |
| Precision | Minimize false positives |
| Recall | Minimize false negatives |
| F1 Score | Balance precision/recall |
| AUC-ROC | Imbalanced classes |
| MSE | Regression |
| MAE | Regression (robust) |
| R² | Regression (explained variance) |

### Ensemble Methods

| Method | Description |
|--------|-------------|
| Bagging | Bootstrap + aggregate (Random Forest) |
| Boosting | Sequential weak learners (XGBoost) |
| Stacking | Meta-learner on base predictions |
| Blending | Similar to stacking, simpler |

### Feature Engineering

| Technique | Description |
|-----------|-------------|
| Scaling | Normalize/standardize features |
| Encoding | Convert categorical to numeric |
| Imputation | Handle missing values |
| Feature creation | Domain-specific features |
| Feature selection | Remove irrelevant features |

### Dimensionality Reduction

| Method | Type |
|--------|------|
| PCA | Linear, unsupervised |
| t-SNE | Non-linear, visualization |
| UMAP | Non-linear, faster |
| LDA | Linear, supervised |
| Autoencoders | Non-linear, neural |
