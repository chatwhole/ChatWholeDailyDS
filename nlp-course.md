## Key Concepts Summary

### Text Preprocessing

| Technique | Description | Example |
|-----------|-------------|---------|
| Lowercase | Convert to lowercase | "Hello" → "hello" |
| Remove punctuation | Strip special chars | "hi!" → "hi" |
| Stop words | Remove common words | "the", "is", "at" |
| Stemming | Reduce to root form | "running" → "run" |
| Lemmatization | Reduce to base form (semantic) | "better" → "good" |

### Text Vectorization Methods

| Method | Description | Pros | Cons |
|--------|-------------|------|------|
| Bag of Words | Word counts | Simple | Loses order |
| TF-IDF | Term frequency-inverse document freq | Weights importance | Sparse |
| Word2Vec | Dense word embeddings | Captures semantics | Fixed vocabulary |
| GloVe | Global vectors | Global statistics | Fixed vocabulary |
| BERT embeddings | Contextual embeddings | Context-aware | Expensive |

### NLP Tasks

| Task | Description | Example |
|------|-------------|---------|
| Sentiment Analysis | Classify text sentiment | Positive/Negative |
| NER | Extract named entities | "Apple" → ORGANIZATION |
| POS Tagging | Part of speech labels | "run" → VERB |
| Text Classification | Assign categories | Spam/Not Spam |
| Machine Translation | Translate languages | EN → FR |
| Summarization | Condense text | Long → Short |
| Question Answering | Answer questions | Context + Question → Answer |
| Text Generation | Generate text | Prompt → Response |

### Transformer Architecture

```
Input → Embedding → Position Encoding → 
[Encoder × N] → [Decoder × N] → Output

Encoder: Multi-Head Attention → Feed Forward
Decoder: Masked Attention → Cross-Attention → Feed Forward
```

### Key Models

| Model | Type | Use Case |
|-------|------|----------|
| BERT | Encoder-only | Understanding tasks |
| GPT | Decoder-only | Generation tasks |
| T5 | Encoder-decoder | Translation, summarization |
| RoBERTa | Encoder (improved BERT) | Understanding tasks |
| DistilBERT | Encoder (lightweight) | Edge deployment |
