## Key Concepts Summary

### Text Preprocessing Pipeline

The first step in any NLP project is cleaning raw text into a standardized format.

| Technique | Description | Example |
|-----------|-------------|---------|
| Lowercasing | Convert all text to lowercase | `"Hello World"` → `"hello world"` |
| HTML Removal | Strip HTML tags from web text | `"<p>Hi</p>"` → `"Hi"` |
| Noise Removal | Remove URLs, emails, special chars | `"Visit https://..."` → `"Visit"` |
| Stopword Removal | Remove common words (the, is, at) | `"The cat sat"` → `"cat sat"` |
| Punctuation Removal | Strip punctuation marks | `"Hello!"` → `"Hello"` |
| Number Handling | Replace or remove numbers | `"3 cats"` → `"[NUM] cats"` |
| Whitespace Normalization | Clean extra spaces and tabs | Multiple spaces → single space |

### Text Vectorization Methods

Convert text into numerical representations for machine learning models.

| Method | Description | Pros | Cons |
|--------|-------------|------|------|
| Bag of Words | Word count vectors | Simple, fast | Loses word order |
| TF-IDF | Weighted term frequency | Captures importance | Sparse, large vocab |
| Word2Vec | Dense 300-d embeddings | Captures semantics | Fixed vocabulary |
| GloVe | Global vector embeddings | Uses global statistics | Fixed vocabulary |
| FastText | Subword embeddings | Handles OOV words | Larger model size |
| BERT Embeddings | Contextual embeddings | Context-aware | Computationally expensive |

### NLP Tasks Overview

| Task | Description | Example | Models |
|------|-------------|---------|--------|
| Sentiment Analysis | Classify text sentiment | Positive / Negative | VADER, BERT, GPT |
| Named Entity Recognition | Extract named entities | "Apple" → ORGANIZATION | spaCy, BERT-NER |
| POS Tagging | Label parts of speech | "run" → VERB | spaCy, NLTK |
| Text Classification | Assign categories | Spam / Not Spam | Naive Bayes, BERT |
| Machine Translation | Translate between languages | English → French | T5, NLLB |
| Summarization | Condense text | Long article → short summary | BART, T5, GPT |
| Question Answering | Answer from context | Context + Q → Answer | BERT-QA, GPT |
| Text Generation | Generate text | Prompt → Response | GPT, LLaMA, Mistral |

### Transformer Architecture Overview

The transformer architecture is the foundation of all modern NLP models:

```
Input Tokens → Token Embedding + Position Encoding →
    ┌─────────────────────────────────┐
    │  Encoder (N layers)             │
    │  ┌─────────────────────────┐   │
    │  │ Multi-Head Self-Attention │   │
    │  │ Add & Layer Norm          │   │
    │  │ Feed Forward Network      │   │
    │  │ Add & Layer Norm          │   │
    │  └─────────────────────────┘   │
    └─────────────────────────────────┘
                ↓
    ┌─────────────────────────────────┐
    │  Decoder (N layers)             │
    │  ┌─────────────────────────┐   │
    │  │ Masked Self-Attention    │   │
    │  │ Cross-Attention          │   │
    │  │ Feed Forward Network      │   │
    │  └─────────────────────────┘   │
    └─────────────────────────────────┘
                ↓
         Output Probabilities
```

### Key NLP Models Comparison

| Model | Architecture | Parameters | Best For |
|-------|-------------|-----------|----------|

### Tokenization Methods Comparison

| Method | Vocab Size | Sequence Length | OOV Handling | Used By |
|--------|-----------|----------------|--------------|---------|
| Word Tokenization | Large (100K+) | Short | Poor | NLTK, spaCy |
| BPE | Medium (30K-50K) | Medium | Good | GPT, RoBERTa, LLaMA |
| WordPiece | Medium (30K) | Medium | Good | BERT, DistilBERT |
| SentencePiece | Medium (32K) | Medium | Good | T5, XLNet, LLaMA |
| Character Tokenization | Small (256) | Very Long | Perfect | CharCNN |

## Code Examples

### Basic NLP Pipeline with NLTK

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from collections import Counter

nltk.download('punkt')
nltk.download('stopwords')

text = "Natural Language Processing enables computers to understand human language."

# Tokenization
tokens = word_tokenize(text)
print("Tokens:", tokens)

# Stopword removal
stop_words = set(stopwords.words('english'))
filtered = [t for t in tokens if t.lower() not in stop_words]
print("Filtered:", filtered)

# Word frequency
freq = Counter(tokens)
print("Most common:", freq.most_common(3))
```

### Sentiment Analysis with Transformers

```python
from transformers import pipeline

# Sentiment analysis
classifier = pipeline("sentiment-analysis")
result = classifier("I love learning NLP!")
print(result)
# [{'label': 'POSITIVE', 'score': 0.9998}]

# Named Entity Recognition
ner = pipeline("ner", aggregation_strategy="simple")
entities = ner("Apple was founded by Steve Jobs in California.")
for ent in entities:
    print(f"{ent['word']}: {ent['entity_group']} ({ent['score']:.2f})")
```

### Text Classification with Scikit-Learn

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
from sklearn.pipeline import Pipeline

# Build pipeline
pipeline = Pipeline([
    ('tfidf', TfidfVectorizer(ngram_range=(1, 2))),
    ('clf', LogisticRegression())
])

# Train
train_texts = ["Great product!", "Terrible service", "Love it!", "Worst ever"]
train_labels = [1, 0, 1, 0]
pipeline.fit(train_texts, train_labels)

# Predict
print(pipeline.predict(["This is wonderful"]))
# [1] = positive
```

### Word2Vec Training

```python
from gensim.models import Word2Vec

```

### Fine-Tuning BERT

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification
from transformers import TrainingArguments, Trainer
from datasets import load_dataset

dataset = load_dataset("imdb")
tokenizer = AutoTokenizer.from_pretrained("distilbert-base-uncased")
model = AutoModelForSequenceClassification.from_pretrained("distilbert-base-uncased", num_labels=2)

def tokenize(batch):
    return tokenizer(batch["text"], padding=True, truncation=True)

dataset = dataset.map(tokenize, batched=True)

training_args = TrainingArguments(
    output_dir="./results",
    num_train_epochs=3,
    per_device_train_batch_size=16,
    evaluation_strategy="epoch"
)

trainer = Trainer(model=model, args=training_args,
                  train_dataset=dataset["train"], eval_dataset=dataset["test"])
trainer.train()
```
